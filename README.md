## 项目简介
- 这是一个轻量级的书评网站  
- 用户分为游客和注册用户  
- 游客只能查看书籍以及书的评论与已有评分  
- 登录/注册后的用户可对书进行评论以及添加一些自己喜欢的书  
- 用户可以对自己的评论与评分增删改查  
- 一本书的推荐度是该书所有评分的平均值  
- 一本书显示的内容主要有作者、推荐度、简介等
